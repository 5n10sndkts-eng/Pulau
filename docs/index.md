## Getting Started

### Prerequisites

- Node.js 18+
- npm 9+

### Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

### Key Files

| Purpose   | File                  |
| --------- | --------------------- |
| App Entry | `src/App.tsx`         |
| Types     | `src/lib/types.ts`    |
| Mock Data | `src/lib/mockData.ts` |
| Helpers   | `src/lib/helpers.ts`  |
