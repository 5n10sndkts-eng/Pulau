# Pulau - Project Overview

## Executive Summary

**Pulau** is a premium travel experience builder application focused on Bali vacation planning. It connects travelers with authentic local tours, activities, and hospitality services through seamless digital booking.

## Purpose & Vision

Pulau aims to transform overwhelming vacation planning into a creative, enjoyable composition process. The application embodies three core experience qualities:

1. **Aspirational** - Every interaction evokes wanderlust and excitement
2. **Trustworthy** - Clear pricing, authentic reviews, and transparent operator information
3. **Effortless** - Complex itinerary planning feels simple through intuitive visual organization

## Architecture Pattern

**Component-based SPA with State Management**

The application follows a single-page application architecture with:
- Screen-based navigation managed via React state
- Persistent state via GitHub Spark KV hooks
- Component composition pattern with shadcn/ui
- Client-side routing simulation through state management

## Key Features

1. **Trip Canvas Building** - Visual itinerary builder with real-time pricing
2. **Experience Discovery** - Categorized browsing with smart filtering
3. **Detailed Experience Pages** - Rich multimedia with provider stories
4. **Multi-Step Checkout** - Guided booking with progress indication
5. **Onboarding Preferences** - Personalized recommendations
6. **Trip Management Dashboard** - Booking history and management
